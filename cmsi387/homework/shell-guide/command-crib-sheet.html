Command Description Example Example Description
cd Change Directory $cd repos go to the repos directory
man access the manual $man man displays information about the man command
ls List information about file(s) $ls -l
ps Process status $ps U $USER View Processes Owned by Current User
pwd Print Working Directory $pwd Shows the working directory
cp copy a file $cp -u *.txt newdir Copy all files ending in .txt into the newdir directory, but only if the files do not already exist in the new directory, or if the files being copied are newer
mv move a file $mv hello.py python_examples moves the file hello.py in the current directory to the directory python_examples
mkdir Create new folder(s) $mkdir ~/repos/ creates the directory named repos
cat Concatenate and print (display) the content of files $cat score.txt names.txt > report.txt combines score.txt and names.txt and creates a new file called report.txt
ping Test a network connection $ping -c 5 google.com send 5 packets to google and stop
grep Search file(s) for lines that match a given pattern $grep -i "hello" hello.py case insensitive search for hello in hello.py
