Symbol Description
~ Home Directory
; Command Separator to execute multiple commands on the same line
, Operation Linker to chain multiple arithmetic operations on the same line
$variable Variable Substitution
command > output-file Redirect the standard output (stdout) of command to output-file
command < input-file Redirect the contents of input-file to the standard input (stdin) of command
command < input-file > output-file Redirect the contents of input-file to the standard input (stdin) of command AND redirect the standard output (stdout) of command to output-file
command >> output-file Append stdout of command to output-file
command << input-file Append stdin of command to output-file
command1 | command2 Redirect stdout of command1 to stdin of command2
pattern$ Regex: matches lines that end with the given pattern
pattern* Regex: matches zero or more instances of the given pattern
pattern{n} Regex: matches exactly n instances of the given pattern
